Experiment information

Description:

The Freesound Taxonomies experiment is a listening test in which you are asked to categorize a number of sounds into a set of "categories". These categories belong to a new taxonomy that we are designing to aid the sound description process in Freesound. The purpose of the experiment is to understand whether or not our taxonomy is clear and understandable for a general audience, and how suitable it is for being integrated with Freesound.

Instructions:

  • The experiment consists of the following 4 phases:
    1. Get familiar with the taxonomy and its categories. There are 5 top-level categories and between 3 and 10 sub categories for each top-level category.
    2. Provide some basic demographic data (e.g. if you are Freesound user, a musician, etc.).
    3. Select the most appropriate category for each of the ~20 sounds that you'll be presented one by one.
    4. Provide any additional textual feedback about the taxonomy (optional).
  • The experiment takes approximately 15-20 minutes. We suggest completing it in a single sitting.
  • It is preferable to be located in a quiet environment and use headphones so you can listen to the sounds without interferences.
  • To ensure accurate results, please refrain from using the browser's back button during the experiment.
